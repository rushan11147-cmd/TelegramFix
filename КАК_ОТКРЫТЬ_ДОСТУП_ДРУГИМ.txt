ВАРИАНТ 2: Локальная сеть (только для друзей рядом) 🏠
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Если друзья в той же WiFi сети:

1. Узнайте ваш IP адрес:
   
   ipconfig
   
   Найдите "IPv4 Address" (например: 192.168.1.100)

2. Запустите Flask:
   
   start_flask_only.bat
   

3. Друзья могут открыть:
   
   http://192.168.1.100:8080
   

⚠️ Работает только пока ваш компьютер включен
⚠️ Только для людей в той же WiFi сети

ВАРИАНТ 3: Ngrok (временный доступ) ⏱️
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Для быстрого тестирования:

1. Скачайте ngrok: https://ngrok.com/download
2. Запустите Flask: start_flask_only.bat
3. В другом терминале: ngrok http 8080
4. Скопируйте URL (например: https://abc123.ngrok.io)
5. Отправьте друзьям этот URL

⚠️ Ссылка работает только пока ngrok запущен
⚠️ Бесплатная версия имеет ограничения
