# 🎨 Новый игровой дизайн

## Дата: 27 января 2026

---

## ✨ Что нового:

### 🎮 Игровой интерфейс
- **Карточки уровней** с красивыми градиентами и тенями
- **Визуальные прогресс-бары** для денег и энергии
- **Карточка персонажа** с аватаром и статистикой
- **Объемные кнопки** с эффектами нажатия
- **Игровые текстуры** (пергамент, дерево, металл)

### 🎨 Дизайн элементов:

#### Верхняя панель
- Темный градиент (синий → серый)
- День и деньги на одной линии
- Иконка денег 💰

#### Карточка уровня
- Фон: бежевый пергамент с градиентом
- Рамка: коричневая деревянная
- Эффект света сверху
- Тени для объема

#### Прогресс-бары
- Деньги: зеленый градиент
- Энергия: оранжевый градиент
- Белая полоска прогресса с свечением
- Цифры по краям

#### Карточка персонажа
- Темный фон (серый градиент)
- Круглый аватар с рамкой
- Информация о работе и черте
- Статистика (зарплата, энергия)

#### Кнопки
- **Работать**: зеленый градиент, большая
- **Навигация**: коричневые (работа), синие (магазин), оранжевые (достижения)
- **Следующий день**: фиолетовый градиент
- Все с тенями и эффектом нажатия

#### Карточки достижений
- Бежевый фон для невыполненных
- Зеленый фон для выполненных
- Иконка в круге
- Награда справа

## 🎯 Особенности дизайна:

### Цветовая схема:
- **Фон**: синий градиент (небо)
- **Карточки**: бежевый пергамент
- **Рамки**: коричневое дерево
- **Кнопки**: яркие градиенты (зеленый, синий, оранжевый)
- **Текст**: темно-серый на светлом, белый на темном

### Эффекты:
- **Тени**: для объема (box-shadow)
- **Градиенты**: для красоты (linear-gradient)
- **Анимации**: при нажатии (transform: translateY)
- **Свечение**: на прогресс-барах

### Адаптивность:
- Максимальная ширина: 450px
- Работает на телефонах
- Все элементы масштабируются

## 🛠 Технические детали:

### CSS:
- Градиенты: `linear-gradient(135deg, ...)`
- Тени: `box-shadow: 0 4px 15px rgba(0,0,0,0.3)`
- Скругления: `border-radius: 15px`
- Анимации: `transition: all 0.2s`

### JavaScript:
- Fetch API для запросов
- Async/await для асинхронности
- Event listeners для кнопок
- Динамическое создание элементов

### Структура:
- Модульный код
- Разделение HTML/CSS/JS
- Переиспользуемые компоненты
