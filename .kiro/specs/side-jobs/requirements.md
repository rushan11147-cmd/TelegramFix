# Требования: Система побочных подработок

## 1. Обзор

Система побочных подработок позволяет игрокам выполнять разовые задания для дополнительного заработка между основной работой.

## 2. Функциональные требования

### 2.1 Типы подработок

**Критерии приемки:**
- Система поддерживает минимум 15 различных типов подработок
- Каждая подработка имеет: название, описание, оплату, затраты энергии, шанс успеха
- Подработки разделены на категории: физические, умственные, творческие, социальные

### 2.2 Ежедневное обновление

**Критерии приемки:**
- Каждый день доступны 3-5 случайных подработок
- Список обновляется при переходе на новый день
- Выполненные подработки исчезают из списка

### 2.3 Выполнение подработки

**Критерии приемки:**
- Игрок может выполнить подработку если хватает энергии
- При выполнении списывается энергия
- Результат определяется случайно с учетом шанса успеха
- При успехе игрок получает деньги
- При провале игрок теряет энергию без оплаты

### 2.4 Влияние навыков

**Критерии приемки:**
- Навык "Удача" увеличивает шанс успеха подработок
- Навык "Харизма" увеличивает оплату социальных подработок
- Навык "Интеллект" увеличивает оплату умственных подработок
- Навык "Скорость" снижает затраты энергии

### 2.5 Интерфейс

**Критерии приемки:**
- Кнопка "Подработки" в главном меню
- Модальное окно со списком доступных подработок
- Отображение: название, описание, оплата, энергия, шанс успеха
- Кнопка "Выполнить" для каждой подработки
- Визуальная индикация успеха/провала

## 4. Нефункциональные требования

### 4.1 Производительность
- Генерация списка подработок < 100мс
- Выполнение подработки < 200мс

### 4.2 Балансировка
- Средняя оплата подработки: 400-500₽
- Средние затраты энергии: 10-12
- Средний шанс успеха: 75-80%

### 4.3 Интеграция
- Система интегрируется с существующей системой энергии
- Система интегрируется с системой навыков
- Данные сохраняются в профиле пользователя
