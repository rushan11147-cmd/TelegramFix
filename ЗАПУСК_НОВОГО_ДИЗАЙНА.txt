═══════════════════════════════════════════════════════════════

🔄 СПОСОБ 2: Сделать новый дизайн главной страницей

Если хочешь чтобы новый дизайн открывался по умолчанию:

1. Открой файл app.py

2. Найди строку (примерно строка 350):
   @app.route('/')
   def index():
       return render_template('simple.html')

3. Измени на:
   @app.route('/')
   def index():
       return render_template('game_design.html')

4. Перезапусти сервер:
   Ctrl+C
   python app.py

5. Открой:
   http://localhost:5000/
